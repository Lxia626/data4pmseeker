Directory simu_data/ stores the random sampling experiments of the genuine genome markers with the original genotype files titled as "xxx.geno.csv"

File 20210909_greedy_exhaus2.txt is the summary of solutions obtained by the two algorithm with the last number of each column referring to the mean value of the indicator.
File 20210909_greedy_exhaus2_number.txt stores the number of markers in greedy-PMSs and exhaust-PMSs.
File 20210909_CPE.txt stores CPE values of the PMSs
File 20210909.pdi_info2.txt records the PDI as the first marker, PE and PIC values.
Accordingly, files named with "nosex" stands for those of no gender information.
